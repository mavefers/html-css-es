Submit vs Button

Existen dos tipos de botones para formularios: < input type="submit" name="nombre" > y < button >.

< input type="submit" >

Input que funciona como botón, su uso es especial para formularios para poder enviar información.

Para motificar el texto que aparece en el botón, se debe agregar el atributo value=" ".


< button type="submit" > < / button >

Botón generalizado, se usa para cualquier tipo de caso en el que se necesite usar un botón.
Se puede personalizar para cualquier tipos de uso.
Para ponerle texto al botón, se le agrega el texto dentro de la etiqueta. - < button type="submit" >Texto/Enviar< / button >

Se le agrega el atributo type="submit" para poder usarlo en los formularios. - < button type="submit" > < / button >

Así cumpliría el mismo rol de < input type="submit" >.


Resumen:

  • < input type="submit" >
    • Se usa de forma directa en los formularios, para poder enviar información.

  • < button type="submit" > < / button >
    • Es un botón mucho más flexible, es ideal para cualquier tipo de botón que se necesite en el proyecto.
