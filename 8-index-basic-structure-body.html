Existen dos tipos de etiquetas dentro de "body":

  • Etiquetas contenedoras:

    Las etiquetas contenedoras son etiquetas que van a llevar más etiquetas por dentro, y nos van a ayudar a generar una estructura de nuestra página web, tal y como lo pudimos ver en la anatomía de nuestr página web.


  • Etiquetas de contenido:

    Son las etiquetas que van a llevar consigo el texto, las imágenes, vídeos, animaciones, y cualquier cosa que se vaya a renderizar en nuestro proyecto.
    Algunas de estas etiquetas son:

    • < header > < header / >

      • - La parte del encabezado de nuestra página web.
      - Sección superior de nuestra página web.
