USO DE CSS EN HTML

El primer paso es ligar un archivo CSS a nuestro proyecto, a un archivo HTML, mediante la etiqueta < link />

Conectando HTML y CSS

Existen tres maneras de implementar diseño a nuestro proyecto: CSS externo, CSS interno, CSS en línea.

  1. CSS externo: Mediante la etiqueta < link /> y un archivo CSS externo - (la manera correcta)
    2. Esta es la manera ideal para implementar CSS en nuestro proyecto, ya que se estaría trabajando de manera ordenada, al hacer el diseño del proyecto en un archivo independiente al documento .html principal.

  2. CSS interno: Mediante la etiqueta < style > - (No es una buena práctica)
    3. Es una etiqueta que permite implementar CSS dentro de un archivo HTML. Se usa en algunas cosas, pero no es la mejor práctica, ya que si las lineas de estilos son muchas, el proyecto comenzará a demorar en cargar.
    Solo es permitido cuando se tienen pocas lineas de CSS.

  3. CSS en línea: Estilos CSS embebidos en el HTML / como atributo directo en una etiqueta HTML: style=" " - (Tampoco es una buena práctica)
    4. Se le puede agregar ciertos estilos al proyecto, dentro del HTML, simplemente usando el tributo style=" " en cada etiqueta html.
    Se le puede modificar el color, el tamaño, el espacio de la caja, la posición, el tamaño de la fuente, etc.
    Ejemplo:

    < p style="color: royalblue;" >Soy un texto< / p> - Soy un texto
    < p style="font-size: 30px;" >Soy un texto< / p> - Soy un texto
    < p style="font-size: 30px; color: royalblue" >Soy un texto< / p> - Soy un texto

    Evitar esta práctica, porque por error podremos reescribir todos los estilos almacenados en nuestro archivo CSS que está dentro de la etiqueta < link />.

Etiqueta < link />

Nos ayuda a referenciar donde está el archivo de CSS que tenemos que jalar.

Como buena práctica, es mejor dividir los archivos en directorios independientes, por un lado el .html y por el otro, el .css

Agregar estilos dentro de la etiqueta < style > < / style >

Esta etiqueta se coloca, únicamente, dentro del apartado del < head >, en la parte superior del código. En esta manera de implementar CSS a nuestro proyecto, tenemos que indicarle al CSS el elemento que queremos modificar.
La sintaxis es generando "brackets" {...} que funcionarían como parámetro de cada elemento que queramos modificar, agregando por dentro, los estilos que queremos implementar en cada elemento. Por dentro, nosotros podremos agregar todos los estilos que queramos.

Ejemplo:
p {
color: red;
font-size: 30px;
}

Si no precisamos la etiqueta en la que queremos aplicar el estilo, usando atributos como id="" o class="", el estilo afectará a todas las etiquetas que coincidan con lo que hemos declarado. Esto quiere decir que, si ponemos estilo a una etiqueta < p >, afectará a todas las etiquetas de párrafo que hayan en el documento, a menos que la etiqueta tenga un identificador/id, < p id="parrafo1" >, o una clase/class, < p class="parrafo-principal" >.

Anatomía del codigo CSS

Selectores: - universal(*) - elemento - .class - #id

Tenemos 4 maneras de mandar a llamar ciertas propiedades para implementar los estilos

  1. Mediante el Selector universal (*)
    2. Selecciona todos los elementos del HTML.

    * {
    atributo: valor;
    atributo: v4l0rpx;
    }
    Selector de etiqueta - (cubre todos los elementos de forma global)


  2. Mediante el elemento/tag
    3. Quiere decir, por la etiqueta a la cual se le quiere agregar un estilo personalizado

    selector {
    atributo: valor;
    atributo: v4l0rpx;
    }
    Selector de etiqueta - (cubre todos los elementos de forma global)


  3. Mediante su clase / Atributo class=" "
    4. Primero que nada, la etiqueta a la que queremos implementar el estilo tiene que tener una clase ( class=" " ) como atributo.
    Para poder mandar a llamar a esa clase que se ha creado como atributo, dentro de la etiqueta seleccionada, se tiene que iniciar escribiendo un punto ( . ) y el nombre de esa clase (class=" "), o sea el valor de la clase que hemos designado (class="valor").

    .selector {
    atributo: valor;
    atributo: v4l0rpx;
    }
    Selector de clase - (class=" ")

    En HTML es posible agregar una misma clase a más de un elemento.


  4. Mediante su identificador / id=" "
    5. Este caso es similar, ya que la etiqueta a la que queremos implementar el estilo tiene que tener un identificador ( id=" " ) como atributo.
    Para poder mandar a llamar a ese identificador que se ha creado como atributo, dentro de la etiqueta seleccionada, se tiene que iniciar escribiendo el signo numeral ( # ), y el nombre del identificador (id=" "), o sea el valor del identificador que hemos designado (id="valor").
    El valor del atributo id=" " puede ser usado solo una vez por página, esto quiere decir que el valor tiene que ser único por página, o sea, que solo puede haber un id="texto1" por documento.

    #selector {
    atributo: valor;
    atributo: v4l0rpx;
    }
    Selector id - (id=" ")

¿En qué momento se debe usar id, y en qué momento se debe usar clase (class=" ")

El atributo class=" " se usa cuando se quiere aplicar un estilo en un grupo selecto de elementos, por ejemplo, cuando hay 5 párrafos pero a solo 3 les quiero aplicar el estilo; para esto, se tiene que agregar el atributo class=" ", con el mismo valor, class="textos" a cada uno de los elementos, los cuales se quiere modificar.

El atributo id=" " se usa cuando se quiere aplicar estilos solo a un único elemento, solo a uno. No se debe colocar el mismo id a más de un elemento.

Colores en los bordes de CSS

La manera en la que se implementan los colores en los bordes en CSS, son:
Cuando se declara un solo color en una sola linea de código border-color: color1, el color afecta a todos los lados.
Cuando se implementan dos colores, border-color: color1 color2;, no hay necesidad de separar con valores con algún signo; en este caso, el primero aplica al TOP y al BOTTOM, y el segundo aplica a los lados, izquierdo y derecho. Cuando se declaran tres colores, el primero aplica en el TOP, el segundo aplica en los lados, izquierdo y derecho, y el segundo aplica al BOTTOM.
