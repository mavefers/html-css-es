Intro CSS

CSS significa Cascading Style Sheets - Significa que es un documento que aplica todos los estilos de forma de cascada.
"cascading" es el nombre del algoritmo que aplica el navegador al momento de estar implementando todos los estilos.

Es la tecnología para estilizar nuestro proyecto, aplicando colores, aplicando la parte de diferentes dimensiones/tamaños, posición, espacio e incluso para mostrar todos los elementos de HTML, no como cajas, sino como círculos o demás figuras.
Con CSS le podemos dar un estilo personalizado a cada uno de nuestros proyectos.
