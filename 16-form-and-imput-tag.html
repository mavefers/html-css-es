Etiquetas form e input

Formularios:

Generar un formulario es una responsabilidad muy importante, ya que es la forma en la que los usuarios se quedan con nosotros, con nuestro producto, o se van.

El mejor formulario es el que no existe.

Esta es la forma erronea de realizar un formulario


Forma correcta de realizar un formulario:

Etiqueta < form action=" " > < / form>

La caja que almacenará todos los elementos que compongan el formulario.
El elemento HTML form (< form >) representa una sección de una página web que contrendrá un formulario, un espacio interactivo entre el usuario y la web, donde se le permitirá al usuario enviar información a un servidor web.

Estos son los elementos para realizar un formulario:
Se usa la etiqueta < form >, que contrendrá otros elementos, como < label >, < input > y < spam >, etc.

label < label for=" ">
Sirve para ponerle un título que representa otro elemento de interfaz de usuario. Estos elementos se asocian poniendo el mismo valor en el atributo for=" " y en el valor del atributo id=" ".
Aquí se coloca el título o texto de representación del elemento < input /> con el que será enlazado, mediante for=" " (< label for=" ">) y id=" " (< input id=" " />).

input < input type=" " id=" " />
El input sirve para crear una entrada de datos del usuario.

Para vincular ambos elementos, el valor del atributo for=" " tiene que ser el mismo que el valor del atributo id=" ".

Existen varios tipos de elemento < input type=" " />.

< input type="submit" />
Genera un botón para enviar los datos recaudados del formulario, a un "manejador de formularios".
El "manejador de formularios" se especifica como valor del atributo action=" " del elemento/etiqueta < form />.
El "manejador de formularios" suele ser una página de servidor, página externa con un script para procesar los datos de entrada.
Para editar el texto del botón que viene por defecto, "Enviar" o "Send", se agrega un atributo llamado value=" ", por donde se ingresará la palabra elegida como su valor. Ejemplo: value="Enviar"

Atributos extras:

id="identificador"
Como su nombre lo indica, es un identificador de elementos, este atributo se puede usar con cualquier etiqueta, y dos etiquetas NO deberían tener el mismo id, el valor del id="valor" es único para cada etiqueta. El id es útil cuando se está trabajando con CSS o JavaScript, ya que te sirve para seleccionar elementos que serán únicos al momento de trabajar.

for=" "
Sirve para indicarle al elemento < label > que va a estar relacionado con el < input /> que tenga el mismo valor de su id=" ". Por eso mismo, dos o más elementos no deberían tener el mismo id.
El atributo for="" solo se utiliza en las etiquetas < label >.

< p > y < span >
Se usa < p > para textos cortos, medianos o largos (Como una descripción, todo un parrafo, etc.) y si quieres agregar estilos a una palabra o sección en especifico usas < span >
Ejemplo: < p > Nunca Pares de Aprender con < span > Platzi < / span > < / p >.
