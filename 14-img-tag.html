Etiqueta 'img'

Datos sobre la etiqueta < img />

Atributos de la etiqueta < img />

src=" "
Atributo que sirve para declarar la ruta, link o dirección de la imagen que pensamos colocar.

alt=" "
Atributo para colocar una información breve de la imagen, para mostrarle al cliente, en caso no se realice una conexión exitosa con el lugar en donde se encuentra la imagen, y no se logre cargar en nuestra web.
Recomendación: Colocar bien la ruta de la imagen.
Es importante para el tema de accesibilidad
Cuando se crean las rutas de archivos o carpetas que vamos a usar, no se deben usar espacios ( ), porque el navegador no los va a entrender. Para reemplazar los espacios, se utiliza un guión "-" o guión bajo "_".

Descargar imágenes: Pexels.com

Cuando se manipule la herramienta online, Pexels.com, para descargar imágenes, se recomienda descargar solo entre estas tres opciones, Grande, Mediano y Pequeño, y luego se comprime manipulándolas con las websites para compresión de imágenes.
