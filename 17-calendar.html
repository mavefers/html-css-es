Tipos de inputs para armar un calendario (de esta manera tendremos el control total de los datos que vamos a recaudar)

< input type="week" name=" " id=" " >
Genera un campo de entrada para seleccionar el número de semana que va en todo el año del que se ha elegido. -

< input type="month" name=" " id=" " >
Genera un campo de entrada para seleccionar el mes que está dentro de tal año. -

The (name=" ") function like a reference for management the info.
El nombre (name=" ") funciona como una referencia para la gestion de la información, se encarga de almacenar todos los datos recaudados, para que luego sean procesados.

El valor del atributo name=" " funcionaría como una variable, y su valor vendrían a ser los datos que los usuarios estarían ingresando.
Esto nos es útil para poder controlar su valor y enviar los datos de cada input, al backend.


La forma más sencilla y rápida de generar un calendario

< input type="datetime-local" name=" " id=" " >
Permite generar un campo de entrada de fecha y hora, dependiendo del navegador. Al utilizar esta etiqueta, no tendremos control total sobre todos los datos, unitariamente, en comparación a como lo haríamos si realizamos el calendario creando generadores de datos (inputs) de manera independiente. -
