Etiqueta 'figure'

< figure > < / figure >
Es una etiqueta contenedora, que está creada para almacenar una o varias imágenes con la etiqueta 'img'.
Sirve para poder definir una dimensión y posición a 'figure', y automáticamente a la etiqueta imagen, < img / >.
Su uso es parte de las buenas prácticas, y semánticamente, es lo correcto para el navegador.

< figcaption > < / figcaption >
Esta etiqueta sirve mostrar al pié de la imagen, una muy breve descripción, autor, o algo importante que haga referencia a la imagen.
Es una descripción similar a lo que vendría a tener la etiqueta 'alt', pero que, visualmente, pueda ayudar a explicar sobre la imagen que estamos viendo, en caso sea necesario.
Va dentro de la etiqueta contenedora < figure > < / figure >.
*Sustituye el uso indebido, mala práctica, de la etiqueta < div >< / div > junto con la etiqueta de párrafo < p > < / p >.
